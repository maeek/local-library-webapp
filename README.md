# Public file library
Browse PUBLIC files, watch videos, listen to music, all that from your browser.
![Local Library](https://raw.githubusercontent.com/maeek/local-library-webapp/master/src/assets/library.png)
## Project setup
```
npm install
```
## Run 
```
npm run start
```

Default port 3000
